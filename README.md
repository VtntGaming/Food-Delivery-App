# Lưu ý khi tạo nhánh và tạo PR
- Tạo nhánh tên là feature-featureName
- Tạo PR vào develop, KHÔNG TẠO PR VÀO MAIN

- Có thể tạo nhiều nhánh, nhưng lưu ý nhỡ kỹ nhánh của mình là nhánh nào.
- Xoá nhánh nếu không còn làm nữa
